🚀 Сборка APK для Telegram Proxy

⚠️ Проблема: На вашем компьютере Python 3.15, который несовместим с buildozer. Используйте один из методов ниже.

🌟 Метод 1: Google Colab (РЕКОМЕНДУЕТСЯ)

Время: 5 минут | Сложность: Легко

  1. Откройте Google Colab
  2. Загрузите файл build_on_colab.ipynb
  3. Загрузите main.py и buildozer.spec через панель Files
  4. Запустите все ячейки (Runtime → Run all)
  5. Скачайте готовый APK
✅ Не требует установки ничего на ваш компьютер!

🐙 Метод 2: GitHub Actions

Время: 10 минут | Сложность: Легко

  1. Создайте репозиторий на GitHub
  2. Загрузите все файлы проекта
  3. Перейдите в Actions → Build APK
  4. Нажмите "Run workflow"
  5. Скачайте APK из Artifacts

🐧 Метод 3: WSL2 (Linux на Windows)

Время: 20 минут | Сложность: Средне

  1. Откройте PowerShell от администратора
  2. Выполните: wsl --install
  3. Перезагрузите компьютер
  4. Установите зависимости в Ubuntu
  5. Запустите buildozer android debug

🐍 Метод 4: Python 3.10

Время: 15 минут | Сложность: Средне

  1. Скачайте Python 3.10
  2. Установите (отметьте "Add to PATH")
  3. Создайте виртуальное окружение
  4. Установите buildozer
  5. Запустите сборку
💡 Совет: Используйте Google Colab - это самый быстрый способ!

📦 Файлы проекта:

  • main.py - код приложения
  • buildozer.spec - конфигурация
  • build_on_colab.ipynb - для Colab
  • README.md - документация
