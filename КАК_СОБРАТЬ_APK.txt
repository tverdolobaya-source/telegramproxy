⚠️ ПРОБЛЕМА:
На вашем компьютере Python 3.15, который несовместим с buildozer.
Buildozer работает только с Python 3.8-3.11.

═══════════════════════════════════════════════════════════════
  ✅ САМЫЙ ПРОСТОЙ СПОСОБ: Google Colab (5 минут)
═══════════════════════════════════════════════════════════════

1. Откройте: https://colab.research.google.com/

2. Нажмите "Файл" → "Загрузить блокнот"

3. Загрузите файл: build_on_colab.ipynb

4. В левой панели нажмите на значок папки 📁

5. Загрузите файлы:
   - main.py
   - buildozer.spec

6. Нажмите "Среда выполнения" → "Запустить все"

7. Подождите 10-15 минут

8. APK автоматически скачается!

═══════════════════════════════════════════════════════════════
  🐙 СПОСОБ 2: GitHub Actions (автоматическая сборка)
═══════════════════════════════════════════════════════════════

1. Создайте аккаунт на https://github.com (если нет)

2. Создайте новый репозиторий (кнопка "+")

3. Загрузите все файлы из этой папки

4. Перейдите во вкладку "Actions"

5. Нажмите "Build APK" → "Run workflow"

6. Подождите 10-15 минут

7. Скачайте APK из раздела "Artifacts"

═══════════════════════════════════════════════════════════════
  🐧 СПОСОБ 3: Установить WSL2 (Linux на Windows)
═══════════════════════════════════════════════════════════════

1. Откройте PowerShell от имени администратора

2. Выполните: wsl --install

3. Перезагрузите компьютер

4. Откройте Ubuntu из меню Пуск

5. В терминале Ubuntu выполните:

   sudo apt update
   sudo apt install -y python3-pip git openjdk-17-jdk
   pip3 install buildozer cython==0.29.33
   
6. Перейдите в папку проекта:
   cd /mnt/c/Users/yaros/OneDrive/Рабочий\ стол/TelegramProxy

7. Запустите сборку:
   buildozer android debug

8. APK будет в папке bin/

═══════════════════════════════════════════════════════════════
  🐍 СПОСОБ 4: Установить Python 3.10
═══════════════════════════════════════════════════════════════

1. Скачайте Python 3.10:
   https://www.python.org/downloads/release/python-31011/

2. Установите (отметьте "Add to PATH")

3. Откройте новый терминал и выполните:
   
   py -3.10 -m venv venv310
   venv310\Scripts\activate
   pip install buildozer
   buildozer android debug

4. APK будет в папке bin/
